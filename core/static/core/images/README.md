# Images de la Hero Section

Placez les deux images suivantes dans ce dossier :

1. **`locataire-hero.png`** - Illustration de la locataire (côté gauche)
   - Jeune femme avec cheveux noirs et mèches orange
   - Tenant un smartphone avec une carte affichant une maison
   - Format recommandé : PNG avec fond transparent ou fond adapté au vert olive

2. **`proprietaire-hero.png`** - Illustration du propriétaire (côté droit)
   - Femme confiante tenant une tablette avec dashboard
   - Format recommandé : PNG avec fond transparent ou fond adapté au vert olive

**Dimensions recommandées :**
- Largeur : 400-500px
- Hauteur : 500-600px
- Format : PNG (avec transparence si possible)
- Résolution : 72-150 DPI pour le web

Les images seront automatiquement redimensionnées et positionnées par le CSS.
